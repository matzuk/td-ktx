searchEmojis

suspend fun TelegramFlow.searchEmojis(text: String?, exactMatch: Boolean, inputLanguageCode: String?): Emojis

Suspend function, which searches for emojis by keywords. Supported only if the file database is enabled.

Parameters

text - Text to search for.

exactMatch - True, if only emojis, which exactly match text needs to be returned.

inputLanguageCode - IETF language tag of the user's input language; may be empty if unknown.

Return
Emojis Represents a list of emoji.
