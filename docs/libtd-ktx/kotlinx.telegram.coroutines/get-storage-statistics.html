getStorageStatistics

suspend fun TelegramFlow.getStorageStatistics(chatLimit: Int): StorageStatistics

Suspend function, which returns storage usage statistics. Can be called before authorization.

Parameters

chatLimit - The maximum number of chats with the largest storage usage for which separate statistics should be returned. All other chats will be grouped in entries with chatId == 0. If the chat info database is not used, the chatLimit is ignored and is always set to 0.

Return
StorageStatistics Contains the exact storage usage statistics split by chats and file type.
