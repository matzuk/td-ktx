addFavoriteSticker

suspend fun TelegramFlow.addFavoriteSticker(sticker: InputFile?): Unit

Suspend function, which adds a new sticker to the list of favorite stickers. The new sticker is added to the top of the list. If the sticker was already in the list, it is removed from the list first. Only stickers belonging to a sticker set can be added to this list.

Parameters

sticker - Sticker file to add.
