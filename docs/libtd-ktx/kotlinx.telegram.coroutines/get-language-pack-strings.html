getLanguagePackStrings

suspend fun TelegramFlow.getLanguagePackStrings(languagePackId: String?, keys: Array<String>?): LanguagePackStrings

Suspend function, which returns strings from a language pack in the current localization target by their keys. Can be called before authorization.

Parameters

languagePackId - Language pack identifier of the strings to be returned.

keys - Language pack keys of the strings to be returned; leave empty to request all available strings.

Return
LanguagePackStrings Contains a list of language pack strings.
