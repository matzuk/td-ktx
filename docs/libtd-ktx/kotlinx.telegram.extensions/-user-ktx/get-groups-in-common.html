getGroupsInCommon

open suspend fun User.getGroupsInCommon(offsetChatId: Long, limit: Int): Chats

Suspend function, which returns a list of common group chats with a given user. Chats are sorted by their type and creation date.

Parameters

offsetChatId - Chat identifier starting from which to return chats; use 0 for the first request.

limit - The maximum number of chats to be returned; up to 100.

Return
TdApi.Chats Represents a list of chats.
