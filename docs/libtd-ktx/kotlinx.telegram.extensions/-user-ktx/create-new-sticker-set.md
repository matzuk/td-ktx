sticker set.

### Parameters

end with *&quot;*by*&amp;lt;bot username&amp;gt;&amp;quot;* (*&amp;lt;botUsername&amp;gt;* is case insensitive); 1-64
characters.

`isMasks` - True, if stickers are masks.

`stickers` - List of stickers to be added to the set.
