createNewStickerSet

open suspend fun User.createNewStickerSet(title: String?, name: String?, isMasks: Boolean, stickers: Array<InputSticker>?): StickerSet

Suspend function, which creates a new sticker set; for bots only. Returns the newly created sticker set.

Parameters

title - Sticker set title; 1-64 characters.

name - Sticker set name. Can contain only English letters, digits and underscores. Must end with "by&lt;bot username&gt;&quot; (&lt;botUsername&gt; is case insensitive); 1-64 characters.

isMasks - True, if stickers are masks.

stickers - List of stickers to be added to the set.

Return
TdApi.StickerSet Represents a sticker set.
