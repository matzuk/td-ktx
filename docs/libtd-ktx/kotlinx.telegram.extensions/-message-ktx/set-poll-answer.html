setPollAnswer

open suspend fun Message.setPollAnswer(chatId: Long, optionIds: IntArray?): Unit

Suspend function, which changes the user answer to a poll. A poll in quiz mode can be answered only once.

Parameters

chatId - Identifier of the chat to which the poll belongs.

optionIds - 0-based identifiers of answer options, chosen by the user. User can choose more than 1 answer option only is the poll allows multiple answers.
