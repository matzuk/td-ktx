editSchedulingState

open suspend fun Message.editSchedulingState(chatId: Long, schedulingState: MessageSchedulingState?): Unit

Suspend function, which edits the time when a scheduled message will be sent. Scheduling state of all messages in the same album or forwarded together with the message will be also changed.

Parameters

chatId - The chat the message belongs to.

schedulingState - The new message scheduling state. Pass null to send the message immediately.
