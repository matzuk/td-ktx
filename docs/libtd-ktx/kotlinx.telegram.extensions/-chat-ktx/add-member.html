addMember

open suspend fun Chat.addMember(userId: Int, forwardLimit: Int): Unit

Suspend function, which adds a new member to a chat. Members can't be added to private or secret chats. Members will not be added until the chat state has been synchronized with the server.

Parameters

userId - Identifier of the user.

forwardLimit - The number of earlier messages from the chat to be forwarded to the new member; up to 100. Ignored for supergroups and channels.
